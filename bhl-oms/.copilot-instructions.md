## Critical Rules

1. **pgx v5:** Always cast `::text` for enum and date columns in SELECT
2. **Money:** Use `NUMERIC(15,2)` in DB, never float64 in Go
3. **Timezone:** UTC in DB, convert `Asia/Ho_Chi_Minh` in app
4. **Existing code is source of truth** — specs in `docs/specs/` are targets for NEW code
5. **Response format:** Use `pkg/response/` — `response.OK()`, `response.Err()`, etc.
6. **Error codes:** Reference `docs/specs/ERROR_CATALOGUE.md`
7. **State machines:** Reference `docs/specs/STATE_MACHINES.md`
8. **Business rules:** Reference `docs/specs/BUSINESS_RULES.md`

## QA — Lỗ Hổng Đã Biết (Fix theo thứ tự này, KHÔNG đảo ngược)

### 🔴 LH-03 [CRITICAL] — Silent error ở ATP + credit check
**File:** `web/src/app/dashboard/orders/new/page.tsx`
- Line ~144: `apiFetch('/atp/batch').catch(console.error)` → fail silently → `hasAtpIssue = false` → user submit đơn vượt tồn kho mà UI báo "OK"
- Line ~95: credit info fetch `.catch(console.error)` → `creditInfo = null` → `creditExceeded = false` → bỏ qua check hạn mức

**Fix pattern bắt buộc:**
```typescript
// ❌ KHÔNG BAO GIỜ
apiFetch('/atp/batch').catch(console.error)

// ✅ LUÔN — handleError đã có tại web/src/lib/handleError.ts
apiFetch('/atp/batch').catch(err => {
  handleError(err, { silent: false })
  setAtpStatus('unknown')  // KHÔNG phải null — button Submit auto-disabled
})
```
**Verify:** Tắt backend → form tạo đơn phải show warning + nút Submit disabled.

### 🔴 LH-04 [CRITICAL] — Offline sync vỏ rỗng
**File:** `web/src/app/dashboard/driver/[id]/page.tsx`
- `useOfflineSync` đã mount nhưng `queueOfflineRequest` chưa được gọi ở bất kỳ đâu
- Banner "sẽ đồng bộ" hiện ra nhưng không có gì được ghi → mất dữ liệu im lặng

**Fix pattern bắt buộc cho 3 hàm:**

```typescript
// handleUpdateStop(), handleSubmitEPOD(), handlePayment()
if (!navigator.onLine) {

// ePOD: ảnh binary KHÔNG queue, chỉ queue metadata
```
**Verify:** Tắt 4G → bấm "Đã đến nơi" → bật 4G → DB phải ghi `stop.status = arrived`.

## QA — Quy Tắc Cứng Khi Vibe-Code

**Trước khi viết code, check 3 câu:**
1. Có liên quan tiền không? → dùng `safeParseVND`, KHÔNG `parseFloat`
2. Có API call critical (ATP, credit, gate check)? → KHÔNG `.catch(console.error)`
3. Là driver offline action? → wire `queueOfflineRequest`

**Sau khi viết code, chạy 2 lệnh:**
```powershell
# Trong bhl-oms/
.\check-code-quality.ps1
```
Hoặc thủ công:
```powershell
Select-String -Path "web/src/**/*.tsx","web/src/**/*.ts" -Pattern "\.catch\(console\.error\)" -Recurse
Select-String -Path "web/src/**/*.tsx" -Pattern "parseFloat|parseInt" -Recurse | Where-Object { $_.Line -match "price|amount|total|payment" }
```
Cả 2 phải trả về **0 kết quả** mới được commit.
